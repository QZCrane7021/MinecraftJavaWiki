# 翻译工具百科

## 文档概要与阅读建议

本文档的作用是为想入坑 Minecraft Java 版的玩家提供单人和多人游戏方面的技术信息。

本文档将会讲解以下工具和网站的基本功能：

- Plain Craft Launcher；
- Modrinth；
- Curseforge

请移步 “基本信息” 版块，了解过后，再移步相关的功能版块。
