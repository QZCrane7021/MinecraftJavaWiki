# Minecraft Java 版介绍

!!! info "何为 “Minecraft Java 版”？"
    本百科全篇的 Minecraft Java 版均指由 Mojang AB 制作并直接发行的、由 Java 程序语言驱动的**桌面** Minecraft 版本。由网易代理的基岩版、Java 版均不在考虑范围内，基于 Fold Craft Launcher、Pojav Launcher 等启动器运行在移动设备上的 Java 版 Minecraft 特性不同，本百科无法涵盖。

Minecraft Java 版最早于 2009 年推出开发版本，于 2011 年推出正式版，热度长盛不衰。时至今日，Minecraft 各版本百花齐放，但社区总体上仍倾向于使用 Java 版搭配模组加载器作为主要版本。

## 优势

Java 语言打包而成的 `.jar` 包可轻松反编译，于是 Java 版自然而然成为了插件和模组的天堂。诚然，这样也会带来代码破解和作弊的风险，但社区的问题往往也可以由社区自行消解。

庞大的模组和插件生态，带来了大量的玩家，形成了宽广的生态护城河。

## 劣势

首先，Java 语言天生要慢于基岩版使用的 C++。虽然 C++ 需要自行管理垃圾回收，下限低，但相应地上限也高。

其次，由于年代久远，出于兼容性考量，Java 版至今仍在使用老旧的 OpenGL 图形引擎。这一图形引擎中，所有的渲染指令必须在一个主线程中按顺序提交，无法充分利用上 GPU。当然，最新的 26.2 版本中，Minecraft 支持了 Vulkan 引擎，但要想彻底抛弃 OpenGL，依旧任重道远。

最后，由于 Java 版是买断制，而基岩版有内购市场，从特别长远的角度看，Java 版可能渐渐被官方忽视。当然，至少近五年内，这种可能都微乎其微。
