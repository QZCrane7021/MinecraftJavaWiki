# Minecraft Java 版介绍

本百科全篇的 Minecraft Java 版均指由 Mojang AB 制作并直接发行的、由 Java 程序语言驱动的**桌面** Minecraft 版本。由网易代理的基岩版、Java 版均不在考虑范围内，基于 Fold Craft Launcher、Pojav Launcher 等启动器运行在移动设备上的 Java 版具体特性有所不同，本百科亦无法涵盖。

Minecraft Java 版最早于 2009 年推出开发版本，于 2011 年推出正式版，热度长盛不衰。时至今日，Minecraft 各版本百花齐放，但社区总体上仍倾向于使用 Java 版搭配模组加载器作为主要版本。在社区内，Minecraft 也常常简称为 MC。

!!! info "鸣潮还是 Minecraft？"
    鸣潮公测后，至今仍有人就 MC 这一简称归谁争论不休。有人以 WuWa（Wuthering Waves）这一简称曾经作为抹黑鸣潮的称呼为由，依旧固执地使用 MC 作为简称，也有人在其间调停，认为可以让大写的 MC 归 Minecraft，小写的 mc 归鸣潮，但后者这种和事佬心态显然不可取，因为互联网能既然为了 MC 的简称归属吵起来，就根本不能指望在交流时还区分大小写。

## 优势

Java 语言打包而成的 `.jar` 包可轻松反编译，于是 Java 版自然而然成为了插件和模组的天堂。诚然，这样也会带来代码破解和作弊的风险，但社区的问题往往也可以由社区自行消解。

庞大的模组和插件生态，带来了大量的玩家，形成了宽广的生态护城河。

## 劣势

首先，Java 语言天生要慢于基岩版使用的 C++。虽然 C++ 需要自行管理垃圾回收，下限低，但相应地上限也高。

其次，由于年代久远，出于兼容性考量，Java 版至今仍在使用老旧的 OpenGL 图形引擎。这一图形引擎中，所有的渲染指令必须在一个主线程中按顺序提交，无法充分利用上 GPU。当然，最新的 26.2 版本中，Minecraft 支持了 Vulkan 引擎，但要想彻底抛弃 OpenGL，依旧任重道远。

最后，由于 Java 版是买断制，而基岩版有内购市场，从特别长远的角度看，Java 版可能渐渐被官方忽视。当然，至少近五年内，这种可能都微乎其微。

## 版本迭代

Minecraft 在更新新版本时，旧版本依旧开放下载，结果 MC 玩家也习惯于多版本分立。

### 低版本

本文中的低版本专指 Minecraft 1.12 及以前的版本。低版本早期更新往往想到哪做到哪，很多时候仰仗社区意见和 Notch 本人的想法，例如鸡蛋曾经不可投掷，但是因为一名玩家宣称 “鸡蛋能丢我就生吃 U 盘” 而加入了这个特性。至于具体沿革，可详见 [Minecraft Wiki][mc-wiki]。

[mc-wiki]:https://zh.minecraft.wiki/w/Java%E7%89%88%E7%89%88%E6%9C%AC%E8%AE%B0%E5%BD%95/%E5%BC%80%E5%8F%91%E7%89%88%E6%9C%AC

### 高版本

本文中的低版本专指 Minecraft 1.13 及以后的版本。在 Mojang 被微软收购后，更新逐渐展现出明确集中的主题，并在 1.13 开始达到顶峰：

- 1.13 — 大修海洋群系和机制，水下引入了可观的游泳动作和功能；
- 1.14 — 大修村庄，建筑重制，灾厄村民体系完备；
- 1.16 — 大修下界，新增数种群系和猪灵相关体系；
- 1.17、1.18 — 大修主世界地下，新增繁茂洞穴，地下洞穴扩展。

1.20 及以后，Mojang 更新渐显颓势，最终取消生物投票，并采用小版本更新。小版本也有主题，部分主题体量虽小，但好评甚于 1.20 和 1.21 早期版本，如 “逐梦天空”“铜器时代”“群骑纷争”。

特别注意，自 2025 年末起，Minecraft 的版本命名变为按年份计，例如 2026 年的第一个版本为 26.1，目前的具体稳定版为 26.1.2。官方预计一年总计将会发布四个小版本，模组适配的具体稳定版至少比之前的 1.21.x 要清晰多了。

### 高版本和老资历

不是所有人都认同上述的高版本分类法。在 MC 社区中，一般更习惯称 1.16.5 以上的版本为高版本，但也有些人认为是 1.18.2、1.20.1、1.21.1 甚至是 26.1，而有些老玩家还停留在古早的 1.7.10。

入坑 MC 后，你可能会面临一个其他游戏社区中从未见过的问题：版本 “老资历”。这些玩家口中的 MC 早已半身不遂，海域大改、村庄翻新、地形大修都是大逆不道，试炼密室、骑兵长矛都是食之无味，只有那个天黑了伸手不见五指、战斗只需要连按左键、度过水域要按死空格在水上沉浮的老版本，才有 “童年的味道”，才是 “唯一真神”。这些人可曾玩过任何版本的 MC 都不可知，与其争论完全是对牛弹琴，**置之不理即可**。当然，**同态复仇绝不可取**，千万不要反过来嘲讽那些因为难以割舍某些模组、插件确实停留在老版本的玩家。
