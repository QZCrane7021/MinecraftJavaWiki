# 入坑 Minecraft Java 版 — 基础

## 运行原理

和大多数游戏不同，Minecraft Java 版并没有可执行文件，而是一个 `.jar` 包。`.jar` 包由 Java 字节码打包而成，需要 JRE（Java Runtime Environment，Java 运行时环境）才能执行。而这个 JRE，如果搜索渠道错了，很可能下载到的不是安装程序而是源码包，需要手动放入硬盘某处、添加 PATH 环境变量，相当麻烦，因此 Java 版的入坑门槛相当高。

所幸，市面上有一些第三方的 “启动器”，可以快捷管理、启动 Minecraft Java 版的实例，有些启动器还兼具自动下载、配置 JRE 的功能。

!!! tip "JRE 和 JDK"
    JDK 是 Java Developer Kit，Java 开发者套件的简称。乍一看，玩家根本不需要开发者功能，明显不该下载 JDK。但是，JDK 必然内置一个 JRE，而自 Java 11 版本起，Oracle 公司就不再分发单独的 JRE。考虑到新入坑的玩家一般都会直接选择高版本，而高版本一般都需要 17、21 甚至 25 版本的 Java，几乎可以说，新玩家无需认识 JRE，直接下载配置 JDK 即可。

## 版本选择

### 高版本和老资历

MC 过往的老版本都得以保留，因此 MC 的玩家社群也存在版本差异，从而产生了高版本的称呼。

多高的版本算高版本？这个问题在 MC 社区中众说纷纭。一般来说，1.16.5 以上均可以认为是高版本，但 MC 一直有新玩家入坑，导致 “何为高版本”，有些人认为是 1.18.2，有些人认为是 1.20.1，有些人认为是 1.21.1，有些人甚至认为是 26.1，而有些老玩家还停留在古早的 1.7.10。

但有一点所有玩家都认同：1.13 版本。这个版本中，MC 的代码大规模重构，导致 1.12- 和 1.13+ 出现天堑，它本身声名狼藉，却给新的模组大版本奠定了基础，因此社区玩家普遍认为 1.13 是分水岭。

在入坑 MC 后，你可能会面临一个其他游戏社区中从未见过的问题：版本 “老资历”。这些玩家口中的 Minecraft 早已半身不遂，海域大改、村庄翻新、地形大修都是大逆不道，试炼密室、骑兵长矛都是食之无味，只有那个天黑了伸手不见五指、战斗只需要连按左键、度过水域要按死空格在水上沉浮的老版本，才有 “童年的味道”，才是 “唯一真神”。这些人可曾玩过任何版本的 MC 都不可知，与其争论完全是对牛弹琴，**置之不理即可**，更不要反过来嘲讽那些因为难以割舍某些模组、插件确实停留在老版本的玩家。
